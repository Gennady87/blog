### Полное руководство по работе с Obsidian: от основ до продвинутых функций

---

### Введение

**Obsidian** — это мощный инструмент для управления знаниями, который превращает ваши заметки в интерактивную базу данных. Он сочетает минималистичный интерфейс, гибкость и расширяемость, что делает его идеальным как для личного использования, так и для командной работы. В этом руководстве мы разберём всё: от первых шагов до сложных возможностей, включая интеграции, плагины и автоматизацию.

---

### Часть 1. Основы

#### Что такое Obsidian?
Obsidian — это приложение для работы с текстовыми файлами в формате Markdown. Оно позволяет:
- Создавать заметки и связывать их между собой с помощью внутренних ссылок.
- Работать локально без зависимости от облачных сервисов.
- Настраивать интерфейс и функции под личные нужды.

---

#### Установка и настройка

1. **Скачивание и установка**:
   - Загрузите Obsidian с [официального сайта](https://obsidian.md).
   - Установите приложение на свой компьютер или мобильное устройство.

2. **Создание хранилища**:
   - При первом запуске выберите опцию «Create a new vault» и укажите папку для заметок.
   - Рекомендуется использовать отдельную папку на диске или в облаке (например, iCloud/Google Drive).

   - Перейдите в «Настройки» → «Оформление», чтобы выбрать тему.

   - Настройте горячие клавиши в разделе «Hotkeys» для быстрого доступа к функциям.

#### Основы Markdown

Markdown — это язык разметки, который используется для форматирования текста. Основные элементы:
- **Заголовки**: `# Заголовок 1`, `## Заголовок 2`, `### Заголовок 3`.
- **Списки**:
  - Маркированные: `- Пункт`.
  - Нумерованные: `1. Первый пункт`.
- **Ссылки**: `[Текст ссылки](URL)`.
- **Изображения**: `![Описание](путь_к_файлу)`.

Пример:
```markdown
# Образец заметки
- [ ] Задача
- [x] Выполненная задача
[Ссылка на сайт](https://example.com)
```

Более подробную информацию о языке разметки Markdown вы можете найти в этой статье: [[1. Основы Markdown]]

### Часть 2. Расширенные функции

#### Визуализация данных с помощью графа
- **Граф**: отображает связи между заметками.
  - Откройте «Graph View», чтобы увидеть карту связей.
  - Настройте фильтры, чтобы сфокусироваться на определённых темах.

#### Работа с тегами
- Теги добавляются с помощью `#` (например, `#Задача`).
- В панели тегов (включается через «Настройки» → «Plugins») отображаются все используемые теги.

---

#### Внутренние ссылки
Создание связей между заметками:
- Вставьте `[[Название заметки]]`, чтобы создать ссылку.
- Если заметка не существует, Obsidian предложит её создать.

### Часть 3. Настройка хранилища

#### Организация файлов
- **Папки**: создавайте папки для структурирования данных.

### Часть 4. Плагины и автоматизация

#### Установка плагинов
1. Перейдите в «Настройки» → «Community Plugins».
2. Включите безопасный режим, выберите плагин из списка и установите.

#### Популярные плагины
- **Templater**: для создания сложных шаблонов.
- **Dataview**: отображение данных в виде таблиц и списков.
- **Calendar**: удобное управление ежедневными заметками.

### Часть 6. Работа с базами данных и сложными структурами

#### Dataview: мощный инструмент для работы с данными

**Dataview** — это плагин, который позволяет структурировать заметки и выводить информацию в виде таблиц, списков или графиков. Его можно использовать для создания динамических баз данных из Markdown-файлов.

1. **Установка Dataview**:
   - Включите плагин через «Настройки» → «Community Plugins».
   - После активации Dataview добавляет поддержку запросов в Markdown.

2. **Примеры использования Dataview**:
   - Таблица задач:
     ```markdown
     ```dataview
     table status as "Статус", due as "Срок"
     from "Задачи"
     where status != "Завершено"
     sort due asc
     ```
     ```
   - Список заметок по тегу:
     ```markdown
     ```dataview
     list from #Проект
     ```
     ```
   - Анализ данных:
     Используйте вычисления для подсчёта элементов:
     ```markdown
     ```dataview
     table length(file.name) as "Кол-во символов в названии"
     from "Проекты"
     ```

3. **Советы по оптимизации**:
   - Создавайте метаданные в начале файла:
     ```markdown
     ---
     status: В процессе
     due: 2024-12-31
     ---
     ```

---

#### Использование шаблонов с Templater
Плагин **Templater** позволяет создавать шаблоны с динамическими элементами:
- **Пример создания заметки о встрече**:
  ```markdown
  # Встреча {{tp_date}}
  - Участники: {{tp_placeholder}}
  - Вопросы: {{tp_placeholder}}
  ```
- **Автоматическое добавление текущей даты**:
  ```markdown
  {{tp_date:YYYY-MM-DD}}
  ```

### Основные горячие клавиши в Obsidian

Obsidian поддерживает множество горячих клавиш для ускорения работы с заметками. Вот список наиболее полезных сочетаний:

---

### **Общие действия**
1. **Создать новую заметку**:  
   - **Windows/Linux**: `Ctrl + N`  
   - **macOS**: `Cmd + N`

2. **Открыть существующую заметку**:  
   - **Windows/Linux**: `Ctrl + O`  
   - **macOS**: `Cmd + O`

3. **Закрыть текущую заметку**:  
   - **Windows/Linux**: `Ctrl + W`  
   - **macOS**: `Cmd + W`

4. **Переключить полноэкранный режим**:  
   - **Windows/Linux**: `F11`  
   - **macOS**: `Ctrl + Cmd + F`

---

### **Редактирование текста**
1. **Форматировать текст как заголовок**:
   - **H1**: `Ctrl + 1` / `Cmd + 1`
   - **H2**: `Ctrl + 2` / `Cmd + 2`
   - **H3**: `Ctrl + 3` / `Cmd + 3`

2. **Создать список**:
   - Маркированный: `-` и пробел.
   - Нумерованный: `1.` и пробел.

3. **Вставить ссылку**:  
   - **Windows/Linux**: `Ctrl + K`  
   - **macOS**: `Cmd + K`

4. **Быстрая вставка внутренней ссылки**:  
   - **Windows/Linux**: `Ctrl + [[`  
   - **macOS**: `Cmd + [[`

5. **Добавить жирный текст**:  
   - **Windows/Linux**: `Ctrl + B`  
   - **macOS**: `Cmd + B`

6. **Добавить курсив**:  
   - **Windows/Linux**: `Ctrl + I`  
   - **macOS**: `Cmd + I`

---

### **Работа с файлами**
1. **Создать вложенную ссылку (сразу создать новую заметку)**:  
   - **Windows/Linux**: `Ctrl + Enter`  
   - **macOS**: `Cmd + Enter` (после ввода текста ссылки `[[Название]]`).

2. **Переименовать текущую заметку**:  
   - **Windows/Linux**: `F2`  
   - **macOS**: `Cmd + R`

3. **Открыть граф**:  
   - **Windows/Linux**: `Ctrl + G`  
   - **macOS**: `Cmd + G`

---

### **Управление панелями**
1. **Открыть новую панель**:  
   - **Windows/Linux**: `Ctrl + P`  
   - **macOS**: `Cmd + P`

2. **Закрыть текущую панель**:  
   - **Windows/Linux**: `Ctrl + W`  
   - **macOS**: `Cmd + W`

3. **Переключиться между режимами просмотра (редактор/просмотр)**:  
   - **Windows/Linux**: `Ctrl + E`  
   - **macOS**: `Cmd + E`

---

### **Поиск и навигация**
1. **Открыть глобальный поиск**:  
   - **Windows/Linux**: `Ctrl + Shift + F`  
   - **macOS**: `Cmd + Shift + F`

2. **Поиск в текущей заметке**:  
   - **Windows/Linux**: `Ctrl + F`  
   - **macOS**: `Cmd + F`

3. **Быстрая смена темы (если настроено)**:  
   - **Windows/Linux**: `Ctrl + T`  
   - **macOS**: `Cmd + T`

4. **Перейти к следующей заметке в истории**:  
   - **Windows/Linux**: `Ctrl + Tab`  
   - **macOS**: `Cmd + Tab`

5. **Перейти к предыдущей заметке в истории**:  
   - **Windows/Linux**: `Ctrl + Shift + Tab`  
   - **macOS**: `Cmd + Shift + Tab`

---

### **Работа с плагинами и шаблонами**
1. **Вставить шаблон (Templater)**:  
   - **Windows/Linux**: `Ctrl + Shift + T`  
   - **macOS**: `Cmd + Shift + T`

2. **Открыть настройки**:  
   - **Windows/Linux**: `Ctrl + ,`  
   - **macOS**: `Cmd + ,`

3. **Вызвать командную палитру**:  
   - **Windows/Linux**: `Ctrl + P`  
   - **macOS**: `Cmd + P`

---
